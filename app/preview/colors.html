Want to build your twitter-bootstrap theme faster?

We first developed bootstrap-magic so we could create our themes a lot faster. We developed these features and techniques in bootstrap-magic so that you, too, can become a bootstrap theme master:

1. Awesome typeahead

You don't have to type out variables and Less functions every time. Defined variables (starting with @) and Less functions (like darken(@color,10%)) will automatically appear in a list as you start typing. Prediction + autocomplete = awesome typeahead

3. Contextual and live preview

Change a variable and you can see the result immediately. No scrolling to check the impact of what you changed.

2. Reusable variables

You can specify your main "Brand Colors" with a new set of variables that we have defined. You can base the other colors in your theme off of these main colors. This technique will give you a well coordinated and good looking theme.

4. New and imported variables accepted

If you already have variables defined, you can import them right into boostrap-magic. You can even create your own new custom variables.

5. Colorpicker

The colorpicker tool lets you easily see and choose colors so you can find just the right ones for your theme.
