How to build your theme faster ?

We first developed this bootsrap theme editor for our own purpose to create theme a lot faster. Here are some technics to become a bootstrap theme chef with this tool.

1. Awesome typeahead

Find as you type all defined variables started with an @ and all the Less function as darken(@color,10%).

3. Contextual and live preview

Change variables and you will get the result on the content on the right. No scroll to check what you have changed.

2. Reusable variables

Configure the mains colors and the new variables that we created "Brand Colors" and reuse it in order to have a smooth theme.

4. New and imported variables accepted

You already have set some variables. Import them and you can even custom your own new variables.

5. Colorpicker

Colorpicker to help you to find lovely colors.
